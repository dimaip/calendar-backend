# Форматы дат в таблицах Airtable

Данные о богослужениях хранятся в двух таблицах Airtable: **Переходящие** (подвижный цикл, привязанный к Пасхе) и **Непереходящие** (неподвижный цикл, привязанный к календарю). Обе таблицы имеют одинаковый набор колонок, но отличаются форматом колонки «Дата».

## Общие колонки обеих таблиц

| Колонка | Описание |
|---|---|
| **Дата** | Ключ, по которому запись привязывается к дню (формат отличается, см. ниже) |
| **Язык** | `Рус` или `Цся` (церковнославянский) |
| **Неделя** | Название недели / периода |
| **Святые** | Святые дня (поддерживает теги `#TP0`–`#TP6`, `#SR`, `#NSR`) |
| **Заглавие чтения** | Заголовок для чтений дня |
| **Утреня**, **Литургия**, **Вечерня** | Чтения на соответствующих службах |
| **1-й час**, **3-й час**, **6-й час**, **9-й час** | Чтения на часах |
| **На освящении воды** | Чтения на водоосвящении |
| **Прокимен**, **Аллилуарий**, **Причастен**, **Входной стих**, **Вместо Трисвятого**, **Задостойник**, **Отпуст** | Части литургии |
| **Тропари**, **Кондаки**, **Величания**, **Эксапостиларии**, **Богородичен** | Общие песнопения (+ варианты с суффиксом `-службы`) |
| **Стихиры на Господи взываю**, **Стихиры на стихах**, **Прокимен вечерни**, **Прокимен триоди 1/2** | Части вечерни |
| **Стихиры на хвалите**, **Стихиры после 50 псалма**, **Прокимен утрени**, **Степенны** | Части утрени |

В колонке «Дата» можно указать несколько значений через запятую — тогда одна запись будет применяться к нескольким дням.

---

## Таблица «Переходящие»

### Формат даты

```
неделя;день
```

- **неделя** — номер недели по Пасхе (1 = Пасхальная неделя)
- **день** — день недели: `0` — воскресенье, `1` — понедельник, ... `6` — суббота

### Примеры

| Дата | Значение |
|---|---|
| `1;0` | Пасха (1-я неделя, воскресенье) |
| `1;1` | Светлый понедельник |
| `7;0` | Неделя Всех Святых |
| `49;0` | 49-я неделя по Пасхе, воскресенье |

### Особенности

Для колонки «Литургия» используется составной формат через `;`:
- `апостол;евангелие` — апостольское и евангельское зачало
- При наличии третьего элемента (`апостол;евангелие;доп`) сдвиг евангельского чтения не применяется

Номер недели может корректироваться в зависимости от периода года (Крестовоздвижение, Богоявление и др.).

## Таблица «Непереходящие»

### Формат даты

```
ДД/ММ[оператор][условие][#кратность]
```

Все даты указываются по **старому стилю** (юлианскому календарю).

### Операторы

#### Без оператора — точная дата

```
25/12           → 25 декабря ст.ст. каждый год
```

#### `+` — первый день недели после даты

```
25/12+0         → первое воскресенье после 25 декабря
25/12+6         → первая суббота после 25 декабря
25/12+w         → первый будний день после 25 декабря
```

Дни недели: `0` — воскресенье, `1` — понедельник, ... `6` — суббота. `w` — будний день (пн–пт).

#### `-` — последний день недели перед датой

```
06/01-0         → последнее воскресенье перед 6 января
06/01-6         → последняя суббота перед 6 января
```

#### `~` — ближайший день недели к дате

```
25/12~6         → суббота, ближайшая к 25 декабря
```

#### `#` — кратность

Используется с `+` и `-` для сдвига на несколько недель:

```
25/12-0#1       → воскресенье перед 25 декабря, сдвинутое на 1 неделю назад
```

#### `=` — условие

Дата применяется, только если условие выполнено.

**По дню недели:**

```
25/12=0         → 25 декабря, только если это воскресенье
25/12=6         → 25 декабря, только если это суббота
25/12=w         → 25 декабря, только если это будний день
```

**Отрицание (`!`):**

```
25/12=!0        → 25 декабря, только если это НЕ воскресенье
25/12=!w        → 25 декабря, только если это НЕ будний день
```

**По неделе пасхального цикла (синхронизация с переходящими):**

Формат: `=неделя;день` — условие на основе подвижного календаря. Позволяет синхронизировать непереходящую запись с переходящим циклом без ежегодных ручных правок.

```
15/04=49;0      → 15 апреля, только если это 49-я неделя по Пасхе, воскресенье
08/09=25;0      → 8 сентября, только если это 25-я неделя по Пасхе, воскресенье
15/04=!49;0     → 15 апреля, кроме случая когда это 49-я неделя по Пасхе, воскресенье
```

## Сводная таблица примеров

| Таблица | Дата | Значение |
|---|---|---|
| Переходящие | `1;0` | Пасха |
| Переходящие | `7;0` | Неделя Всех Святых |
| Переходящие | `49;3` | 49-я неделя по Пасхе, среда |
| Непереходящие | `25/12` | Рождество Христово |
| Непереходящие | `25/12=0` | Рождество, если в воскресенье |
| Непереходящие | `25/12=!0` | Рождество, если НЕ в воскресенье |
| Непереходящие | `25/12+0` | Первое воскресенье после Рождества |
| Непереходящие | `06/01-0` | Последнее воскресенье перед Богоявлением |
| Непереходящие | `15/04=49;0` | 15 апреля при совпадении с 49-й неделей по Пасхе |
| Непереходящие | `15/04=!49;0` | 15 апреля кроме совпадения с 49-й неделей по Пасхе |
